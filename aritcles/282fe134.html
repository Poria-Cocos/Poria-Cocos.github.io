VESC 从零开始制作无刷电机驱动器系列（三）

发表于2025-04-30|更新于2025-04-30|VESC
|总字数:669|阅读时长:2分钟|浏览量:

这一章将介绍 VESC TOOL 上位机的主要参数、功能。

笔者会优先介绍重要和常用的部分，几乎不用的部分会省略。

若未特别说明，本文电机转速均指电转速，并非输出轴的实际机械转速。

由于笔者使用的电机驱动IC无通信功能，所以有关故障报错的功能将会省略。

页面各部分介绍

Motor Settings

General

General

  • Invert Motor Direction：修改电机正反向定义
  • Sensor Port Mode：电机旋转情况的反馈传感器选择

Current

  • Motor Current Max：最大驱动电机电流，需考虑电机承受电流能力
  • Motor Current Max Brake：最大制动电机电流，需考虑电池吸收电流能力
  • Absolute Maximum Current：极限电流，超出后会返回错误，通常会设置的尽可能大
  • Battery Current Max：最大电池输出电流
  • Battery Current Max Regen：最大电池回馈电流，需考虑电池吸收电流能力

Voltage

  • Battery Voltage Cutoff Start：电池保护起始电压，低于此电压开始限制输出电流，需根据电池实际电压设置
  • Battery Voltage Cutoff End：电池保护结束电压，低于此电压停止输出电流，需根据电池实际电压设置

RPM

  • Max ERPM/Max ERPM Reverse：电机最大转速，需根据电机数据手册设置

Advanced

  • Minimum Input Voltage：电池欠压保护
  • Maximum Input Voltage：电池过压保护
  • Minimum Duty Cycle：最小驱动占空比
  • Maximum Duty Cycle：最大驱动占空比
  • Minimum Current：最小驱动电流

FOC

General

  • Sensor Mode：电机旋转情况的反馈传感器选择
  • Motor Resistance ®、Motor Inductance (L)、Motor Flux Linkage (λ{\lambda}λ)、Current KP、Current KI：标定出的参数

后面的几个页面是不同的传感器参数标定、修改页面。

PID Controllers

  • Minimum ERPM：最小电机转速，小于此值将不进行转速控制
  • Allow Braking：是否允许制动
  • Ramp eRPMs per second：电机转速爬升率，-1表示无限制。为了安全、减小电调/电池压力，此参数尽量根据实际使用情况给一个限制，不要使用-1
  • Position Controller：进行位置环控制时，PID参数需要用户自行调整

Experiments

  • 进行自定义的电机控制实验

APP Settings

General

  • APP to Use：使用的预设
  • VESC ID：电调的id，多电调配置时必须设置，每个电调的id应不同
  • Can Status Message Mode：CAN返回报文的信息（详细内容见CAN控制篇）
  • Can Status Rate：CAN返回报文的频率
  • CAN Baud Rate：CAN波特率

Data Analysis

Realtime Data

  • 电调实时数据采集、显示
  • 点击最右侧的RT按钮开启功能
